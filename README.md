# News App

## 概要
- News APIから最新のトップニュースを取得して、一覧形式で表示
- 気になったニュースはクリップ機能で保存ができる
- 無限スクロール機能でニュースをサクサク取得できる

## 主な使用技術
- React Native 0.64.3
- TypeScript 4.3.5
- Expo 44.0.x
- React Navigation 6.x.x
- React Redux 7.2.x
- axios 0.24.x
